git 사용법
------------------------------------
git에 파일 올리기 (push)

1. 파일 등록
ex) git add baek/파일명

2. 파일 commit
ex) git commit -m "메세지내용"

3.파일 올리기
ex) git push orgin main
------------------------------------
git 파일 다운로드

1. 파일 다운로드
ex) git pull origin


------------------------------------
브랜치 생성

1. 현재 브랜치 리스트
ex) git branch

2. 브랜치 생성
ex) git branch 브랜치명

3. 브랜치 변경
ex) git checkout 브랜치명

------------------------------------
브랜치 삭제

1. 현재 브랜치 리스트
ex) git branch

2. 브랜치 삭제
ex) git branch -d 브랜치명
